പുതിയ ഇടപാട്

ഇടപാട് ചേർക്കുക

Admin Area: Sheets Sync

എങ്ങനെ സെറ്റപ്പ് ചെയ്യാം:
1. ഒരു പുതിയ Google Sheet ഉണ്ടാക്കുക.
2. Extensions → Apps Script തുറക്കുക.
3. താഴെ "സ്ക്രിപ്റ്റ് കോപ്പി ചെയ്യുക" ബട്ടൺ അമർത്തി code കോപ്പി ചെയ്ത് അവിടെ paste ചെയ്യുക (നിലവിലുള്ളത് മുഴുവൻ മാറ്റി).
4. Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone → Deploy.
5. കിട്ടുന്ന Web App URL ഇവിടെ താഴെ paste ചെയ്യുക, "സേവ് & ടെസ്റ്റ്" അമർത്തുക.
6. അതിന് ശേഷം താഴെ "Users" section വന്നാൽ അഡ്മിൻ യൂസറിനെ admin എന്ന പേരിൽ ചേർക്കുക.
