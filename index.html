ഇടപാട് ചേർക്കുക

Admin Area: Sheets Sync

എങ്ങനെ സെറ്റപ്പ് ചെയ്യാം:
1. പുതിയ മാറ്റങ്ങൾ വർക്ക് ചെയ്യാൻ, നിങ്ങളുടെ Google Sheet-ൽ 'Users' എന്നൊരു ടാബ് ഉണ്ടെങ്കിൽ അത് Delete ചെയ്യുക (Data ടാബ് കളയരുത്!).
2. Extensions → Apps Script തുറക്കുക.
3. താഴെ കോപ്പി ചെയ്ത് പഴയത് മാറ്റി അവിടെ paste ചെയ്യുക.
4. Deploy → New deployment → Web app → Anyone → Deploy.
5. പുതിയ URL ഇവിടെ paste ചെയ്ത് "സേവ് & ടെസ്റ്റ്" അമർത്തുക.
6. താഴെ "+ പുതിയ User ചേർക്കുക" വഴി അഡ്മിൻ യൂസറിനെ admin എന്ന പേരിൽ ചേർക്കുക.
